\doxysubsection{Detailed Description}
Classe GUI servant à la création d\textquotesingle{}un compte Cheque. 

\begin{DoxyVerb}         La classe maintient dans un état cohérent ces renseignements et
         est construit en utilisant le framework Qt.           
         Cette classe contient l'interface pour l'ajout d'un compte Cheque.
         La classe n'accepte qu'un compte Cheque valide.
         C'est la responsabilité de l'utilisateur de la classe de s'en asssurer.  
\end{DoxyVerb}
 \textbackslash{}slot\+Ajout\+Epargne Slot de l\textquotesingle{}interface pour l\textquotesingle{}ajout d\textquotesingle{}un compte Cheque
