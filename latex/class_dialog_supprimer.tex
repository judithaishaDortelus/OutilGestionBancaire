\doxysubsection{Detailed Description}
Classe GUI permettant la suppression d\textquotesingle{}un compte bancaire. 

\begin{DoxyVerb}         Cette classe fournit une interface utilisateur, construite avec Qt, pour saisir
         le numéro d'un compte à supprimer. Elle offre un slot pour valider l'entrée
         et fermer la boîte de dialogue. La suppression effective du compte est gérée
         par la fenêtre principale après la fermeture du dialogue.
\end{DoxyVerb}
 \textbackslash{}slot\+Supprimer Slot déclenché pour valider l\textquotesingle{}entrée et fermer la boîte de dialogue.
